# Android应用权限交互链智能合规分析系统 前端原型

基于 `React + Vite + Tailwind CSS` 的本地可运行前端原型项目，使用静态 mock 数据，不依赖后端服务。

## 启动方式

```bash
cd prototype-ui
npm install
npm run dev
```

默认开发地址：

```bash
http://localhost:5173
```

## 页面说明

- 登录页
- 首页 / 控制台
- 应用导入与设备连接页
- 权限交互链分析页
- 结果展示页
- 历史记录页

## 说明

- 所有页面均为中文展示
- 所有数据均来自 `src/mock/data.js`
- 不依赖任何第三方商业 Logo 或品牌标识
- 适合用于软件著作权说明书截图与演示
